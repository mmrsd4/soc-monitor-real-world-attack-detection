Splunk Alert Setup



Objective



Create alerts for detected attacks.







Steps



1\. Run detection query



2\. Click \*\*Save As → Alert\*\*



3\. Enter:



&#x20;  \* Title: Attack Detection Alert

&#x20;  \* Condition: Number of results > 0

&#x20;  \* Trigger: Per Result



4\. Click Save







Result



\* Alerts generated on detection

\* Enables real-time monitoring
