Dashboard Setup



Objective



Visualize attack data in Splunk.







Steps



1\. Go to Dashboards

2\. Click Create New Dashboard

3\. Add panels







Panels



\* Failed Logins (Line Chart)

\* Top Attacking IPs (Bar Chart)

\* SQL Injection Events (Table)

\* Alerts (Column Chart)







Result



\* Centralized monitoring

\* Visual attack insights
